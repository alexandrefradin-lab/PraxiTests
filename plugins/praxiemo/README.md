# PraxiEmo — Plugin Intelligence Émotionnelle

Évaluation de l'IE en 16 dimensions inspirée des modèles Bar-On EQ-i et Goleman.

## Caractéristiques

- 80 questions Likert 4 (Jamais → Toujours)
- 6 questions de fiabilité (Marlowe-Crowne)
- 16 dimensions regroupées en 4 familles : Conscience de soi, Régulation émotionnelle, Relations & Communication, Leadership émotionnel
- Score global 80-320 → 4 niveaux (Faible, Modéré, Élevé, Très élevé)
- Détection automatique de biais de désirabilité sociale
- Top 3 forces + axes de progression (dimensions ≤ 12/20)

## Origine

Porté depuis WordPress plugin `praxiemo` v1.1.9.
