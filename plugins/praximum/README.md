# PraxiMum — Plugin Big Five

Test de personnalité Big Five (OCEAN) — 30 facettes détaillées avec scoring T normé.

## Caractéristiques

- 128 questions Likert 1-4 (120 OCEAN + 8 désirabilité sociale)
- 5 dimensions : **O**uverture · **C**onscience · **E**xtraversion · **A**gréabilité · **N**euroticisme
- 30 facettes (6 par dimension)
- Scoring T normé (population française, mean/sd par facette)
- Détection biais désirabilité sociale
- 16 étapes de 8 questions (UX rythmée)

## Données

Les 128 questions sont stockées en JSON (`src/Data/questions.json`) — extraction directe du plugin WordPress source.

## Origine

Porté depuis WordPress plugin `plugin-personnalite` (PraxiMum) v1.4.1.
