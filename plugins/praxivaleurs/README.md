# PraxiValeurs — Plugin Valeurs Schwartz

Évaluation des valeurs professionnelles selon le modèle universel de Schwartz.

## Caractéristiques

- 40 questions Likert 1-6
- 10 valeurs : Autonomie · Stimulation · Hédonisme · Réussite · Pouvoir · Sécurité · Conformité · Tradition · Bienveillance · Universalisme
- Scoring Likert pondéré (60%) + tournoi facultatif (40%)
- Top 5 valeurs présenté avec définitions

## Origine

Porté depuis WordPress plugin `praxivaleurs` v3.10.0.
